# Themes

## Snazzy

```lisp
;; Snazzy palette
(green      "#5af78e")
(yellow     "#f3f99d")
(blue       "#57c7ff")
(magenta    "#ff6ac1")
(cyan       "#9aedfe")
(orange     "#ffb86c")

(accent-1 magenta)
(fg-prompt accent-1)

;; code
(keyword orange)
(builtin magenta)
(string yellow)
(constant green)
(fnname blue)
(type cyan)
```
